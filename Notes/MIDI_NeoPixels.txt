Pixels/Notes 0-59 are on Right Arm
Pixels/Notes 60-119 are on Left Arm

-----------------------------------------------------------------------------------------------------

CCs:

Sends values to ALL pixels.  For pulsing, strobing effects of the whole strip.

Channel Number = color.  1=Red, 2=Green, 3=Blue  4=White (white
overwrites other colours, others combine)

CC# = 74 (Brightness)

Value = Brightness (multiply by 2)

-----------------------------------------------------------------------------------------------------

MIDI Notes:

Note ON Turns an individual pixel on.
Note OFF Turns an individual pixel off.

Channel Number = color.  1=Red, 2=Green, 3=Blue  4=White (white
overwrites other colours, others combine)
Note number = Pixel#
Velocity = Brightness (multiply by 2)

-----------------------------------------------------------------------------------------------------

NOTE ON/OFF to Note ????? turns off all pixels.
